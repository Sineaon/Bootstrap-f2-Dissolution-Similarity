# Bootstrap f2 Dissolution Profile Similarity Analysis Template
📖 简介 / Introduction
基于 Bootstrap 法计算 f2 置信区间的 Excel 模板，适用于...
✅ 功能特点 / Features
•
Bootstrap 法估计 f2 下置信限（f2*）
•
支持 5000 次迭代，结果稳定可靠
•
中英双语界面，符合 FDA/NMPA 指南
📥 下载使用 / Download
点击右侧 Releases 或直接下载 .xlsm 文件
⚠️ 使用须知 / Requirements
Windows + Microsoft Excel，需启用宏（VBA）
📚 引用 / Citation
Mendyk A et al. Dissolut Technol, 2013, 20(1): 13–17.
